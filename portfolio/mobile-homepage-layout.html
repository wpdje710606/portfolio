Project: Mobile NAV and Homepage Buttons

To refresh the mobile appearance of the Site Header and Brand Store, we will be updating the Navigation and Category Buttons.

To improve the customer experience on the mobile view, the Navigation and Category buttons will be redesigned. This update will involve conducting research, creating mockups, implementing the changes, testing them, and finally launching the updated design. Additionally, this update will serve as the template for the entire brand store.

Brand Store

Elements: Icons, Searching bar and Navigation List
Goal: Enhanced website accessibility for seamless navigation.
Process:

Following the website's standard and style, I redesigned the icons in SVG format to maintain quality. For the search bar placement, after exploring various ideas, it was decided to position it below the logo and icons, with the length matching the size of the mobile view device. The manager will determine which elements are displayed directly and which are hidden within the collapsible user interface.

Elements: Primary Collapse and Secondary Collapse
Goal: Separating from the header to infuse more of the brand's style into the brand store.
Process:

Extensive research was conducted for this update as having two navigations on the same mobile page is uncommon. The challenge was to make the brand store Navigation stand out within limited space. Several mockups were created, and the final design includes two/three category collapse buttons along with a hamburger menu that hides the remaining options. Additionally, a standardized CSS template was developed for consistency across all brand stores.
